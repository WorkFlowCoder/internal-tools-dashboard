# internal-tools-dashboard

Tools management dashboard built with React and Tailwind CSS. Provides monitoring of tools, costs, and usage through a dashboard, tools catalog, and analytics pages.

## Installation & Lancement

Installer les dépendances

```Bash
npm install
```

Lancer le serveur de développement

```Bash
npm run dev
```

Lancement des tests

```Bash
npm test
```

## Stack Technique

## Structure du Projet

- **components** : Composants réutilisables (Cards, Modals, Grilles).

- **pages** : Vues principales (Dashboard, Tools, ToolDetail).

- **hooks** : Logique personnalisée pour les appels API.
